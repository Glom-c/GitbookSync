# 🔆 内联函数

## 什么是内联函数？

内联函数是基于函数的动作式语言，有着易懂的语法与优越的性能，可以轻松实现诸多功能\
通过简单易懂的API，无论是**脚本开发者**还是**java/kotlin开发者** 都可以快速自由地使用/拓展内联函数

相较于 **JavaScript** 其有着更低的门槛 兼顾 高自由度 与 较低门槛

{% hint style="info" %}
您可以将内联函数**嵌入**到你开发的插件中，以达到更高自由度
{% endhint %}
